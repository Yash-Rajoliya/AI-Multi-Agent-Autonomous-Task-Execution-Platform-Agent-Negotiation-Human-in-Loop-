# Agent System Design

## Agent Types

- Planner → creates plan
- Researcher → gathers data
- Executor → performs actions
- Critic → evaluates output
- Reflection → improves strategy

---

## Cognition Loop

1. Observe
2. Think
3. Act
4. Reflect

---

## Tool Usage

Agents dynamically select tools based on context.

---

## Memory Integration

- Working memory → short-term
- Semantic memory → knowledge
- Episodic memory → history
